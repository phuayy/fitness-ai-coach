# Deployment Notes

This repository is a local MVP starter. For production:

- Serve frontend and backend over HTTPS.
- Restrict CORS origins.
- Add authentication.
- Add structured logs and request IDs.
- Add rate limits and peer limits.
- Add TURN server support for restrictive networks.
- Store only consented, minimal session data.
- Prefer on-device inference for a future mobile-native version.
